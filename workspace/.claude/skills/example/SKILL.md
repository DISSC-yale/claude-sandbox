This is a placeholder skill. To create your own:

1. Create a new folder under `.claude/skills/` with your skill name
2. Add a `SKILL.md` file with frontmatter (`name`, `description`) and instructions
3. Invoke it with `/your-skill-name` in the Claude prompt
